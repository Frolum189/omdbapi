# Пітч

Легкий та швидкий вебзастосунок для кіноманів та звичайних глядачів, який допомагає швидко знайти цікавий фільм на вечір та дізнатися його сюжет, рейтинг і дату виходу без зайвої реклами чи реєстрацій.

# MVP

Базова версія застосунку, яка повинна виконувати ці функції: користувач може побачити підбірку випадкових новинок на головній сторінці, знайти потрібний фільм за назвою та переглянути детальну картку з його постером і описом.

# Стейкхолдери

1. Основну цінність від сайту отримають звичайні користувачі, які хочуть швидко обирати кіно для перегляду без зайвих складнощів.
2. Оцінювати досягнення цілей MVP будуть перші користувачі (тестувальники), а також викладач під час перевірки та захисту проєкту.
3. На реалізацію та темп розробки впливатимуть мої власні ресурси, мій час як розробника, а також стабільність роботи зовнішнього сервісу OMDb API.

# Мета першої ітерації

Запустити базову версію сайту, де найголовніші функції доступні та працюють коректно, щоб користувач міг успішно знайти потрібний фільм за назвою та побачити його детальний опис.

# Ознаки успіху (Definition of Success)

1. Пошук фільму за назвою працює коректно і швидко видає правильний результат.
2. Картка з постером та інформацією про кіно відображається рівно, а всі дані з API підтягуються без помилок.

# Проблема. 
Людям часто буває важко швидко знайти базову інформацію про фільм або підібрати щось на вечір без перегляду перевантажених рекламою та зайвою інформацією сайтів.
# Ціль. 
Створити швидкий, мінімалістичний та зручний застосунок для миттєвого пошуку кіно та відображення головних деталей про нього.
# MVP. 
Базова версія сайту, де працює головна сторінка і реалізований пошук фільму за точною назвою з виведенням його постера та опису.
# Критерії успіху. 
Пошук працює коректно, дані з сервера підтягуються без помилок, а картка фільму адаптивно відображається на екрані.

# Початковий перелік завдань (Фічі для першого інкременту)
Завдання 1: Створення базової HTML-структури та файлів стилів для головної сторінки.
Завдання 2: Налаштування простої навігації між головною сторінкою та сторінкою пошуку.
Завдання 3: Написання функції на JavaScript для підключення до OMDb API.
Завдання 4: Створення логіки пошуку фільму за назвою та малювання детальної картки з результатами на екрані.
